day 16笔记

1.泛型

  • 形式：< > 一对尖括号

  • 泛型是什么：泛化的类型，或者说是指类型参数

  • 在实现方法的功能时，有些数据时未知的，需要泛型

  • 泛型也有形式参数和实际参数，但是<>中填写的一定是数据类型，而且一定是引用数据类型。

  • <泛型形参>一般用大写字母，例如：

    ​ :type类型，泛指类型

    ​ element,元素类型

    ​ return type,返回值类型

    ​ argument，形参的类型

  • <泛型实参>一般写具体的类型

2. 为什么要使用泛型

避免类型转换，和类型转换错误

3.泛型类与泛型接口

  • 语法形式：

修饰符 class 类名 extends 父类 implements 父接口们{

}
